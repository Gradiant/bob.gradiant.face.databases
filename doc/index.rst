===========================
bob.gradiant.face.databases
===========================

.. todolist::

At present, this plugin provides support for the following databases:

* MSU-MFSD
* OULU-NPU
* REPLAY-ATTACK
* REPLAY-MOBILE
* AllPadDatabases (Aggregated MSU-MFSD, OULU-NPU, REPLAY-ATTACK,  REPLAY-MOBILE)
* UVAD (No tested yet)
